# HospitalDB

Консольное приложение для работы с вымышленной PostgreSQL базой данных HospitalDB.

## Установка необходимых библиотек

```commandline
python -m pip install -r requirements.txt
```

## Запуск приложения

### Из консоли

```commandline
python main.py
```

### Из проводника

Откройте `main.py` с помощью Python.
